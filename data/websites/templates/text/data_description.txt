## Template Guidance

**Primary goal:** Provide background information about data, including
methodology and data sources. This Content Type is not intended to be used to
present complex data, but rather to describe the data.

Can also include information on:

* Types of data
  * Data systems
  * How the data is collected
  * What the data includes
  * How the data is interpreted
  * Data use and impact
  * Data reporting
  * Data quality
  * Data definitions
  * Data management
  * Resources
